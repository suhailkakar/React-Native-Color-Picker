A **color picker** is a graphical user interface widget, usually found within graphics software or online, used to select colors and sometimes to create color schemes. A color picker is used to select and adjust color values. In graphic design and image editing, users typically choose colors via an interface with a visual representation of a colorâ€”organized with quasi-perceptually-relevant hue, saturation and lightness dimensions (HSL) â€“ instead of keying in alphanumeric text values. Because color appearance depends on comparison of neighboring colors (see color vision), many interfaces attempt to clarify the relationships between colors.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/suhailkakar/React-Native-Color-Picker
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   or 
   
     ```sh
   yarn install
   ```
3. Run the app
   ```sh
   npm start 
   ```
   or
     ```sh
   expo start
   ```
