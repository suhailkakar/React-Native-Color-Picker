### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/suhailkakar/React-Native-Color-Picker
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   or 
   
     ```sh
   yarn install
   ```
3. Run the app
   ```sh
   npm start 
   ```
   or
     ```sh
   expo start
   ```
